### 项目方向重大决策：小程序开发

**背景**：用户认为当前项目（微博爬虫 + 管理后台）引流后缺乏落地位置，需要补充用户端小程序。

**讨论内容**：

1. **平台选择**：用户考虑微信/支付宝/抖音三个小程序都做
   - 原因：三个平台用户群体互补（新人+核心玩家）
   - 我的建议：先做微信，验证后再扩展

2. **技术选型**：推荐 **Taro 4.x（React）**
   - 一套代码输出微信/支付宝/抖音三端
   - 用户前端已用 React，上手快
   - 支持 H5 降级

3. **小程序 MVP 规划**：
   - 第一期：术语查询 + 商品推荐 + 分享能力
   - 第二期：微信登录 + 用户画像 + 互动
   - 第三期：会员/订阅（需升级工商个体户后）

4. **支付策略**：
